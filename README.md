# BagNest-Backend
This repository contains the backend logic for the BagNest e-commerce website.  Built using **Express.js** and **JavaScript**, the server provides RESTful APIs to manage products efficiently.  
### Core Functionality

- CRUD operations for products
- JWT-based admin authentication
- Image upload handling with Cloudinary
- Middleware-based route protection
- Error handling and validation

Designed following clean API architecture and production-ready practices.
